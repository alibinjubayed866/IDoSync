# IDOSync Android Application

This repository contains the latest pre-compiled Androi d Application Package (APK) for **IDOSync**.

## Download and Install

1. Download the `IDOSync.apk` file from the main directory or from the Releases page (if available).
2. Transfer it to your Android device (or download it directly from your device's browser).
3. Open the file to install. Note: You may need to enable "Install from Unknown Sources" or "Install Unknown Apps" in your Android settings.
